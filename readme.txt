To Export TDH (Title-tag, meta Description, H-tag) from each pages and posts.

== Description ==

### You can see a TDH list for your site.
You can export the following TDH-tags from each page and post with 1-click.

* Title tag
* meta Description tag
* H tag

### Usage scenarios

For SEO, it is important to implement the TDH tags wisely. However, to see it in action, you need to go to each page and check the html source code. It is very time-consuming and tedious. This plugin solves it with 1-click.

### How to use

1. First install and activate plugin.
1. Go TDH Export Plugin page from admin menu.
1. Select "Post Type" and "Export Method", and then click "Export TDH" buttons.
1. You can see a TDH list for your site.

== Installation ==

1. Upload or install the plugin through your WordPress admin.
1. Activate the plugin through the 'Plugins' admin menu.
1. You can see "TDH Export" in admin menu.

== Frequently Asked Questions ==

= If there are multiple meta Descriptions, which one will be exported? =

The first tag is exported.

= Why do some pages not appear in the TDH list? =

Only public pages will export TDH. If you want to export TDH for non-public pages as well, please publish the pages.

= Why do I get an authentication error? =

If you are using authentication functions on your site, you may not be able to export TDH. In that case, please log in to the site once and try again without closing your browser.

= Where can I report bugs and issues? =
Please report bugs and issues on the plugin's [GitHub page](https://github.com/horisawak/tdh-export/).
